# MyFeed PWA — デプロイ手順

## ファイル構成
```
myfeed-app/
├── index.html     ← メインアプリ
├── manifest.json  ← PWA設定
├── sw.js          ← Service Worker（オフライン対応）
└── README.md
```

## スマホアプリ化の手順

### 1. 無料ホスティングにアップロード（おすすめ：Netlify）
1. https://netlify.com にアクセス
2. 無料アカウントを作成
3. このフォルダをドラッグ＆ドロップするだけ
4. 自動でURLが発行される（例：myfeed-abc123.netlify.app）

### 2. iPhoneでホーム画面に追加
1. SafariでそのURLを開く
2. 画面下の「共有」ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. アプリアイコンとして追加完了！

### 3. Androidでホーム画面に追加
1. ChromeでURLを開く
2. 「ホーム画面に追加」バナーが表示される
3. またはメニュー →「アプリをインストール」

## 機能
- AIカテゴリ分類（AI・ビジネス・テック・マーケット）
- キーワード検索・フィルタ
- AI要約（Claude API使用）
- 記事保存・ブックマーク
- ネイティブ共有（スマホのシェアシートを使用）
- オフライン対応（Service Worker）
- セーフエリア対応（ノッチ・ホームインジケーター）

## カスタマイズ
index.html の `ARTICLES` 配列を編集するか、
RSSフィード取得処理を `loadFeed()` 関数に追加してください。
